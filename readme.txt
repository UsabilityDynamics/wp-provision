== Description ==

WordPress site provisioning.

= Features =

* Resize images on the client-side.

== Installation ==

1. Upload all the files into your wp-content/plugins directory, be sure to put them into a folder called "wp-provision"
2. Activate the plugin at the plugin administration page
3. Follow set-up steps on main Settings > Provisioning page
